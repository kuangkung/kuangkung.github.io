7 本项目中的 gRPC 接口调用描述以及说明

Modifier and Type Method and Description
Boolean registerModel(String id)
获取模型在 gRPC 服务端注册成功与否的结果
id 为算法管理页面添加模型成功时自动生成并插入数据库的UUID。
返回 true 表示注册成功， false 表示注册失败。
ModelCallResult callGearWaveform(Map<String, WaveformPac> map, String wheelDiameter)
获取大齿轮波形，map 需要一个振动波形和一个转速波形，温度波形可选。
wheelDiameter 为车轮直径。
返回值为 ModelCallResult 对象。
ModelCallResult callPinionWaveform(Map<String, WaveformPac> map, String wheelDiameter)
获取小齿轮波形，参数同 callGearWaveform(…)，返回值为 ModelCallResult 对象。
ModelCallResult callBoxWaveform(Map<String, WaveformPac> map, String wheelDiameter)
获取箱体波形，map 需要两个振动波形和一个转速波形，温度波形可选。
wheelDiameter 为车轮直径。
返回值为 ModelCallResult 对象。
WaveformPac callSpeedWaveform(Map<String, WaveformPac> map, String wheelDiameter)
获取速度波形，map 需要一个转速波形。
wheelDiameter 为车轮直径。
返回值为 WaveformPac 对象。
ModelRecall experimentRecall(Map<String, WaveformPac> map, String wd, String expId)
重演试验过程中获取指定算法输出的试验波形。
map 需要四个振动波形，四个温度波形和一个转速波形。
wd 为车轮直径，expId 为新建该重演试验时生成的 UUID。
返回值为 ModelRecall 对象。
